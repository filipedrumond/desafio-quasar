# Desafio de Quasar
Lembre-se de rodar a Fake-API 

## Install the dependencies
```bash
npm install
```

## Run fake API JSON-SERVER
```bash
npm run runDB
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```

### Lint the files and fix
```bash
npm run lint
```

### Build the app for production
```bash
quasar build
```
